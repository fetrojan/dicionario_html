Dicionário de HTML

Este é um exemplo prático que explica 10 diferentes tags HTML.

Tag de Título (h1)

A tag <h1> define o título principal da página.

Tag de Parágrafo (p)

A tag <p> é usada para definir um parágrafo de texto.

Tag de Imagem (img)

A tag <img> é usada para inserir imagens na página.

Tag de Link (a)

A tag <a> é usada para criar hyperlinks que levam a outras páginas.

Tag de Lista Ordenada (ol)

A tag <ol> é usada para criar uma lista ordenada.

Tag de Lista Não Ordenada (ul)

A tag <ul> é usada para criar uma lista não ordenada.

Tag de Item de Lista (li)

A tag <li> é usada para definir um item dentro de uma lista.

Tag de Cabeçalho (header)

A tag <header> representa o cabeçalho de uma seção ou página.

Tag de Rodapé (footer)

A tag <footer> representa o rodapé de uma seção ou página.

Tag de Vídeo (video)

A tag <video> permite incorporar vídeos na página.
